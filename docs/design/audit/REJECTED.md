# Rejected capture set

The three PNG files in this folder were captured before the Windows capture process and Tk application shared the same DPI-awareness mode. At 125% display scaling, the saved images omitted the lower and right portions of the actual window.

These files are retained only as troubleshooting evidence. They are not valid product-audit screenshots and must not be used for UI, responsive-layout, or accessibility claims. The accepted current-build evidence lives in `../verification/` after visual inspection.
